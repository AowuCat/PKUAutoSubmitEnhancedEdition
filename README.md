## 思路
通过ADB获取手机上的健康宝和行程码截图，传到电脑上。
selenium+win32api实现自动填写和上传图片。
里面的点位仅适用于我的手机和当前桌面，需要自行修改。还需要自行添加健康宝和行程码的各个阶段的图片，通过简单的像素对比判断当前阶段，算是能用吧。
之前没接触过玩机，肯定有很多不足之处，仅供参考。
所得健康宝和行程码均通过正常渠道，真实有效。不自动p图（牢饭警告）。
